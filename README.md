# NoteBetter Studio

[English](https://github.com/lineside0418/Notebetter-Studio/blob/main/README_EN.md) | 日本語

**NoteBetter Studio** は、Minecraft Fabric Mod "**NoteBetter**" のための設定ファイルを、ブラウザ上で直感的かつ簡単に作成できるツールです。
ブロックを叩いた際の楽器（Instrument）、音程（Pitch）、音量（Volume）を視覚的に管理し、必要なJSONファイルを自動生成します。

## ✨ 主な機能

- **視覚的な編集**: ブロックのアイコンを確認しながら、直感的な操作が可能です。

- **カスタムサウンド対応**: バニラのサウンドだけでなく、独自のサウンドイベント設定もサポートしています。

- **リソースパック支援**: カスタムサウンド利用時に必要な sounds.json の記述内容を自動生成する機能を搭載しています。

- **ダークモード搭載**: 作業環境に合わせて外観を切り替えられます。

## 🛠️ 使用方法

1. **起動**: [Webサイト](https://lineside0418.github.io/NoteBetter-Studio/)にアクセスします。

2. **プロジェクトの開始**:

    - 「新規プロジェクト」: 新しく設定を作成します。

    - 「インポート」: 既存の notebetterfabric.json を読み込み、編集を再開します。

3. **ブロックの追加**:

    - 画面右下の 「＋」ボタン をクリックします。

    - 設定したいブロックを検索して選択します（例: glass, diamond_block）。

    - リストに存在しないMod等のブロックは、「+ Add Custom」 からIDを直接入力して追加可能です。

4. **音の設定**:

    - 追加されたブロックをクリックして選択します（複数選択も可能です）。

    - 画面右側の インスペクターパネル で以下の項目を調整します：

        - Sound Event: 再生するサウンド（例: minecraft:block.note_block.harp）。検索機能も利用できます。

        - Pitch: 音の高さ（0.5 〜 2.0）。

        - Volume: 音の大きさ。

        - Custom Sound: リソースパックが必要な独自のサウンドイベントを使用する場合に有効にします。

5. **書き出し (Export)**:

    - 画面右上の 「書き出し」 ボタンをクリックします。

    - Config: Mod用設定ファイル (notebetterfabric.json) をダウンロードまたはコピーし、.minecraft/config/ フォルダに配置してください。

    - Resource Pack: カスタムサウンドを使用する場合、ここから sounds.json の内容を取得できます。必要なフォルダ構成のガイドも表示されます。

## 💻 動作環境

- モダンブラウザ（Google Chrome, Edge, Firefox, Safari 等）

- インターネット接続

    - 初回読み込み時にブロックのテクスチャ画像を外部サーバーから取得するために必要です。アプリケーションの動作自体はローカルで行われます。

## ⚠️ 注意事項

- 本ツールは非公式のファンツールです。

- ブラウザのキャッシュクリア等により一時保存データが消失する可能性があるため、こまめに「設定をDL」を行い、バックアップを取ることを推奨します。
